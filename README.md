# ISMO Landing Page - Luxury Tech v1.0

Landing page premium diseñada para **Sistemas ISMO**, enfocada en la automatización de procesos y gestión de negocios con una estética de alta tecnología ("Luxury Tech").

## 🚀 Características

- **Diseño Inmersivo:** Fondos dinámicos, partículas RGB y efectos 3D.
- **Carrusel 3D:** Galería de sistemas gamificada con visualización holográfica de flyers.
- **Arquitectura Dinámica:** Sección de giros de negocio con cambios de tema cromático en tiempo real.
- **Optimización Mobile:** Totalmente responsiva y fluida.
- **Interacción Premium:** Cursor personalizado con efecto LED y animaciones de hardware.

## 🛠️ Tecnologías

- HTML5 / CSS3 (Vanilla CSS + Tailwind CSS CDN)
- JavaScript (Animaciones custom, 3D Transforms)
- Google Fonts (Inter, Outfit)

## 📦 Despliegue

Esta landing page está lista para ser desplegada en **Vercel** o **Netlify** simplemente subiendo los archivos a un repositorio de GitHub.
